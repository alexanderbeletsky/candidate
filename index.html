Candidate - Continuous Deployment app

the shortcut between Visual Studio solution and deployed web application

  • Step 1: Create new site

    Simply create new web site you want setup for continues deployment

  • Step 2: Configure site for deployment

    Run through the configuration including Github, Visual Studio and IIS settings

  • Step 3: Start the deployment

    Run application for initial deployment

  • Step 4: Application launched

    As soon it's done, application is ready to be accessed

  • Step 5: Setup Github hook

    Setup Github post hook, so each time you "git push" to github your application automatically redeployed!
